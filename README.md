# 🛍️ Simple E-Commerce Web App
This is a fully functional frontend-only e-commerce store built using HTML, CSS, and vanilla JavaScript. The application fetches products from the DummyJSON API, supports user authentication, search, sorting, cart with quantity controls, checkout, and a light/dark mode toggle – all with no external frameworks or libraries.

> Designed and developed by [Dhanushkavi](https://github.com/Dhanushkavi)

---

## 🚀 Features

- 🔐 **Login & Register System** (localStorage-based)
- 🌓 **Light/Dark Theme Toggle** (saved in localStorage)
- 🛒 **Dynamic Cart System** with quantity `+/-` buttons
- 🧾 **Checkout Page** with user form validation
- 🔎 **Search Products** by title or description
- 🔃 **Sort by Price** (low to high, high to low)
- 📱 **Fully Responsive** for desktop, tablet & mobile
- ⚙️ **LocalStorage-powered** for cart & auth persistence

## 🧠 How It Works

1. Register → Login → Browse Products
2. Add products to cart using dynamic `+/-` buttons
3. View cart and proceed to checkout
4. Place order (mocked) and return to homepage

---

## 🛠️ Technologies

- HTML5, CSS3, JavaScript (ES6+)
- DummyJSON API (Product list)
- Responsive Flexbox + Media Queries
- LocalStorage (for cart & auth)
